# Infanino
A programming language for maintainable code.

## Features
* Object Orientation
* Structurally typed
* Declarative
* Strict simplicity
* No global scope

## Transpilers

We provide transpilers that converts Infanino code into another language. <br />
So far, the convertible languages are:
* Javascript
